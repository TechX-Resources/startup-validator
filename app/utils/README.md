# app/utils

**Purpose:** Shared helper functions (logging, formatting, retries, etc.) used across the app.

**What students will implement:**
- **logger.py:** Configure a simple logger (e.g. Python logging) so all modules can log to console or file.
- **helpers.py:** Any shared helpers (e.g. truncate text for LLM, safe JSON parse, retry decorator). Start minimal; add as needed.

**Weeks:** 1–2 (logger early), then add helpers as you build.
